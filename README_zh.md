# Deepdance Prompt Review — 终端 AI 视频提示词框架

一个可在多种终端 AI Agent / CLI / Skill 系统中复用的视频提示词框架，专为 **Seedance 2.0 / 即梦** 的中文视频生成工作流设计。

它把自然语言创意转成带生成前审核、模式路由、多模态引用、时长策略和 Deepdance 欲望张力模式的结构化中文提示词，可直接复制到即梦平台使用。

## 框架定位

- **通用框架**：可复制到任何支持 `system prompt`、`skill` 或长指令注入的终端 AI 工具。
- **Claude Code adapter**：仓库内置 `.claude/skills/seedance/SKILL.md`，可按 Claude Code skill 方式安装。
- **Codex adapter**：可同步到 `~/.codex/skills/seedance-prompt-lite/SKILL.md`，保留轻量一键出 prompt 风格。

## 适用终端

- Claude Code
- Codex
- OpenClaw / Hermes / Artemis 等支持本地 skill 或 instruction 的 agent CLI
- 任何可粘贴长 `system prompt` 的终端 AI 工具

## 功能特性

### 十大核心提示词生成能力

| # | 能力 | 说明 |
|---|------|------|
| 1 | **纯文本生成** | 无需任何参考素材，纯靠文字描述生成视频 |
| 2 | **一致性控制** | 通过上传参考图片，保持角色/产品/场景在多镜头间的统一 |
| 3 | **运镜与动作复刻** | 上传参考视频，精准复刻其中的镜头语言、复杂动作和节奏变化 |
| 4 | **创意模板/特效复刻** | 照着参考视频模仿创意转场、广告成片、电影片段 |
| 5 | **剧情创作/补全** | 根据分镜图片或脚本自动生成完整剧情 |
| 6 | **视频延长** | 对已有视频进行前向或后向平滑延长 |
| 7 | **声音控制** | 音色参考、对白生成、音效设计 |
| 8 | **一镜到底** | 生成连贯长镜头，从一个场景平滑过渡到另一个场景 |
| 9 | **视频编辑** | 基于已有视频进行角色替换、剧情颠覆、元素增减 |
| 10 | **音乐卡点** | 让画面节奏与音乐节拍精准匹配 |

### 高级技巧

- **时间戳分镜法** — 按秒精确控制 15 秒视频的每个镜头（如 `0-3秒：...`、`4-8秒：...`）
- **技术参数指定** — 在提示词开头声明画幅比、帧率、色调、镜头风格
- **禁止项声明** — 明确排除不需要的元素（水印、字幕、Logo 等）
- **超长视频分段拼接** — 自动将超过 15 秒的视频拆分为多段，通过视频延长串接，并附带衔接点描述
- **Deepdance 欲望张力模式** — 将爱、欲望、克制、沉沦等抽象概念转译为带时间戳的伪纪录片粗粝感、第四面墙机位和高反差视觉分镜

### 生成前审核机制

Skill 在生成提示词前会先给出 `审核结果：通过 / 需补充 / 需改写 / 拒绝`，用于提前发现平台拦截风险、授权风险和 prompt 质量问题。该机制只作为生成前 guardrail，不替代即梦/Seedance 平台最终审核。

审核覆盖：

- **平台硬限制**：时长 4-15 秒、超长视频分段延长、素材数量限制、`@图片/@视频/@音频` 引用编号。
- **真人与权利风险**：写实真人脸素材、真人授权、声音复刻、名人/IP/品牌高拟真复刻。
- **高风险内容**：未成年人伤害、自伤鼓励、仇恨歧视、血腥虐待、诈骗/误导。
- **质量自检**：主体、动作、镜头、声音、负面约束、连续性和可复制性。

说明：Skill 不设置成人色情/裸露的专门审核规则；仅在涉及未成年人、非自愿真人、违法行为或平台硬限制时处理。

### 多模态引用系统

支持 Seedance 2.0 的 `@` 引用语法，组合多种输入模态：

- **图片**（`@图片1` ~ `@图片9`）— 角色参考、场景背景、首帧/尾帧控制
- **视频**（`@视频1` ~ `@视频3`）— 运镜、动作、特效、节奏参考
- **音频**（`@音频1` ~ `@音频3`）— 背景音乐、音色、音效参考

### 场景化提示词策略

- **电商/广告** — 产品 360 度旋转展示、爆炸分解、3D 渲染特效
- **短剧/对白** — 剧本化场景，含角色台词、情绪标注和音效设计
- **仙侠/奇幻动画** — 电影级武打、法术特效、变身变装序列
- **科普教学** — 4K 医学 CGI、半透明人体结构可视化
- **MV/音乐卡点** — 宽银幕构图、多图卡点参考视频节奏
- **Deepdance / 爱与欲望** — 纪实冲动 POV、赛博洛可可迷离、剥离与留白三种高张力映射

### 内置提示词素材库

Skill 内置了专业的词汇库：

- **镜头语言** — 景别、运镜、角度、节奏、焦点、转场
- **视觉风格** — 画面质感、影像风格、色调氛围、艺术风格、光影效果、动画风格
- **欲望张力词库** — 手持呼吸晃动、直视镜头、极近微距、汗水、衣料摩擦、霓虹明暗交界和材质反差

## 核心框架

- **生成前审核 gate**：先输出 `审核结果：通过 / 需补充 / 需改写 / 拒绝`，再决定生成、追问、改写或拒绝。
- **模式路由**：在纯文本、角色一致性、运镜复刻、视频延长、编辑、音乐卡点、Deepdance 等模式中选择最合适的一类。
- **多模态引用规范**：统一使用 `@图片1`、`@视频1`、`@音频1`，并写清每个素材用途。
- **时长策略**：4-15 秒直接生成，超过 15 秒使用首段生成 + `将@视频1延长Xs`。
- **Deepdance 欲望张力模式**：使用纪实冲动 POV、赛博洛可可迷离、剥离与留白三套视觉映射。
- **输出合同**：普通提示词、参考素材、长视频和 Deepdance 都有固定结构，保证可复制、可验证。

## 使用方法

### 快速开始

在任意已加载该框架的终端 AI 工具中描述你想要创建的视频：

```text
帮我生成一段赛博朋克风格的城市夜景视频提示词，8秒，16:9
```

框架会先判断审核结果和模式，再输出可直接复制到即梦平台的中文提示词。若缺少关键参数或存在平台/授权风险，它会先追问、改写或给出安全替代方向。

### 触发方式

在支持自动 skill 触发的工具里，以下表达会触发该框架：

- 使用 `/seedance` 命令
- 提到关键词：`Seedance`、`即梦`、`视频提示词`、`视频生成`、`AI视频`、`短剧`、`广告视频`、`视频延长`
- 提到 `Deepdance`、`爱与欲望`、`伪纪录片`、`导演场控摘要`
- 要求“生成视频提示词”或“写视频描述”

### 交互流程

框架遵循结构化工作流：

```
第零步：生成前审核 → 输出 审核结果
第一步：识别模式 → 普通 Seedance / Deepdance / 延长 / 编辑 / 卡点等
第二步：补齐参数 → 时长、比例、素材映射、连续性要求
第三步：生成提示词 → ≤15秒直接生成，>15秒输出多段拼接方案
第四步：微调优化 → 调整时间段画面、镜头语言、台词、音效或风格
```

## Seedance 2.0 平台规格

| 维度 | 规格 |
|------|------|
| 图片输入 | jpeg/png/webp/bmp/tiff/gif，最多 9 张，单张 <30MB |
| 视频输入 | mp4/mov，最多 3 个，总时长 2-15 秒，单个 <50MB，分辨率 480p-720p |
| 音频输入 | mp3/wav，最多 3 个，总时长 ≤15 秒，单个 <15MB |
| 文本输入 | 自然语言描述 |
| 文件上限 | 最多 12 个文件（图片 + 视频 + 音频合计） |
| 生成时长 | 4-15 秒，可自由选择 |
| 声音输出 | 自带音效/配乐 |
| 分辨率 | 支持 2K 输出 |

> **重要提示：** 平台不支持上传含有写实真人脸部的素材（图片和视频均不可），系统会自动拦截。

## 最佳实践

- **描述要具体有画面感** — "穿红风衣的女人走过雨后霓虹闪烁的街道"远比"一个女人在走路"效果好
- **13-15 秒视频使用时间戳分镜** — 按秒划分镜头，精确控制每段画面内容
- **台词和音效单独描述** — 与画面描述分行书写，结构更清晰，生成效果更好
- **结尾添加禁止项** — 如"禁止出现水印、字幕、Logo"
- **参考图片风格要匹配主题** — 例如古风主题用水墨风格图片，赛博朋克主题用科幻 CG 图片
- **超过 15 秒的视频** — 交给 Skill 自动拆分为多段，每段之间会附带明确的画面衔接点描述
- **台词用引号包裹** — 并标注角色名称和情绪（如"总裁，卑微慌乱"）
- **情绪氛围不要忽略** — 情绪和氛围的描述对最终效果影响很大

## 常见问题

### Skill 没有自动激活怎么办？

如果使用 Claude Code adapter，确保 `SKILL.md` 文件放置在 `.claude/skills/seedance/SKILL.md`。如果仍然无法激活，尝试直接使用 `/seedance` 命令触发。其他终端工具请确认已把该文件内容注入为 system instruction 或 skill body。

### 生成的提示词效果不理想？

- 尝试提供更具体的描述，避免过于抽象的表述
- 使用时间戳分镜法精确控制每段画面
- 在提示词结尾添加禁止项，排除不想要的元素
- 让 Claude 生成多个版本，挑选最合适的进行微调

### 支持英文提示词吗？

目前 Skill 专为中文提示词优化，因为 Seedance 2.0 平台对中文自然语言理解能力最强。生成的所有提示词（包括视频提示词和配套的图片生成提示词）均为中文。

### 超长视频如何保证分段之间的连贯性？

Skill 会在每段提示词后附带「衔接点」描述，精确描述该段结尾的画面状态。下一段的开头会自然承接上一段的结尾，通过 Seedance 2.0 的视频延长功能实现平滑过渡。
